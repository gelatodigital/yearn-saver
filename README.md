This repo contains smart contracts for the community Yearn Saver

1. Clone this repo
2. Create `.env` fule
```
cp .env.example .env
```
3. Put your Infura ID in .env
4. yarn install
```
yarn install
```
5. Run tests
```
npx buidler test
```
